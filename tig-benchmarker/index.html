TIG Browser Benchmarker

Instructions:

  1. Open the browser console
  2. Setup: `await benchmarker.setup(<api_url>, <api_key>, <player_id>)`
  3. Start: `await benchmarker.start(<num_workers>)`
  4. Stop: `await benchmarker.stop()`
  5. Select Algorithm: `await benchmarker.select_algorithm(<challenge_id>, <algorithm_id>)`
  6. View State: `await benchmarker.state()`
